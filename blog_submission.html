The Program

The following Python program loads the Gapminder dataset, applies a series of data management decisions, and outputs organized, labeled frequency distributions for three variables: Continent, GDP Category (a secondary variable derived from gdpPercap), and Life Expectancy Group (a secondary variable derived from lifeExp). 

import pandas as pd
import numpy  as np

# ── 1. Load dataset ────────────────────────────────────────
url = ("https://raw.githubusercontent.com/resbaz/r-novice-gapminder/"
       "main/data/gapminder-FiveYearData.csv")
df = pd.read_csv(url)
print(f"Dataset loaded: {df.shape[0]} rows × {df.shape[1]} columns")

# ── 2. Data Management Decisions ───────────────────────────

# 2a. Remove rows with missing key values
df = df.dropna(subset=["lifeExp", "gdpPercap", "pop"])

# 2b. Create secondary variable: GDP category
def gdp_category(val):
    if   val < 1000:  return "Low (<$1k)"
    elif val < 5000:  return "Lower-Middle ($1k–5k)"
    elif val < 15000: return "Upper-Middle ($5k–15k)"
    else:             return "High (>$15k)"

df["gdp_category"] = df["gdpPercap"].apply(gdp_category)

# 2c. Create secondary variable: Life Expectancy group
def life_group(val):
    if   val < 50: return "Low (<50)"
    elif val < 65: return "Medium (50–64)"
    elif val < 75: return "High (65–74)"
    else:          return "Very High (75+)"

df["life_group"] = df["lifeExp"].apply(life_group)

# 2d. Select relevant columns only
df = df[["country","continent","year","lifeExp",
         "gdpPercap","pop","gdp_category","life_group"]]

# ── 3. Frequency Distributions ─────────────────────────────
for col in ["continent", "gdp_category", "life_group"]:
    freq = df[col].value_counts()
    pct  = df[col].value_counts(normalize=True).mul(100).round(1)
    tbl  = pd.DataFrame({"Count": freq, "Percent (%)": pct})
    print(f"\n[{col.upper()}]")
    print(tbl.to_string())
    print(f"Missing values: {df[col].isnull().sum()}")

Summary of Frequency Distributions

Data management decisions included removing rows with missing key values, creating two secondary categorical variables by binning continuous measures, and selecting only the eight most relevant columns for analysis.

Continent is the most unevenly distributed variable. Africa dominates with 624 country-year records (28.5%), reflecting that it has the most countries in the dataset. Oceania has only 24 records (1.1%), covering just Australia and New Zealand. No missing values were found for this variable.

GDP Category is a secondary variable created by binning the continuous gdpPercap measure into four meaningful economic tiers. The distribution is relatively balanced: Lower-Middle income countries appear most often (28.4%), while High-income countries are represented nearly as often (23.4%). This suggests the dataset spans a broad range of economic development without overrepresenting wealthy nations. No missing values were present.

Life Expectancy Group is a secondary variable binned from lifeExp into four ordinal categories. The majority of observations fall in the "High (65–74)" range (33.5%), and only 19.1% of records reflect life expectancies below 50 years — largely concentrated in earlier decades and African nations. Approximately 19.1% of observations show very high life expectancies (75+), generally corresponding to wealthier, post-2000 records. No missing values were present after the data cleaning step.
